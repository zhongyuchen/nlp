## 1. 正则表达式


1. \d 数字
2. \w 字母、数字、下划线（不包括汉字）
3. \s 空白符
4. . 除了换行符以外的任何字符
5. [abc] 匹配abc中的

相反的：
1. \D
2. \W
3. \S
4. [^abc] 

量词：
1. *就是{0,}
2. +就是{1,}
3. ?就是{0,1}
4. {n}匹配前一个字符n词
5. 上面四种后面加上？就是指尽量少重复（在量词后加问号？表示尽可能少的匹配，一 旦条件满足就再不往下匹配）
6. 默认情况下，量词都是贪婪模式(greedy quantifier),即匹配到下一个字符不满足匹配规则为止

其他
1. `-` 不需要转义
2. ^表示开头，[^]中的^表示除了……以外的字符
3. $结尾
4. 在[]中的* + ?等元字符都是被当作其本身
5. [\s\S]、[\w\W]、[\d\D]都可以表示任意字符

优先级
```
\                          转移符
() (?!) (?=) [ ]               括号、字符组、环视
* + ? {n} {n,} {n,m}          量词
^ $                       起始结束位置
|                        选择
```
